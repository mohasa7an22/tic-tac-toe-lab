# Description
this is a tic tac toe game made by using html css and javascript, the game has a feature saying whether it is player's X turn or O, it resets as soon as it detects a tie and runs smoothly with no issues.
# Technologies used
HTML is the main front end language which i used to develop the structure of the game.

CSS was used to adjust the style of the game itself and make it more see-able and less of an eyestrain and it was also used to adjust the placement of the tic tac toe board and made the box with the grid using CSS combined with HTML

JavaScript was the language that made the entire game it was basically the brain of the program which made all the functionality happen
